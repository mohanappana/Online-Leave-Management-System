## Technologies Used
<b> Frontend: </b> HTML, CSS, Bootstrap, JavaScript(JQuery) </br>
<b> Backend: </b> Python Flask server </br>
<b> Database: </b> MySQL relational database </br>

## Features
<b> 1. Efficient: </b>Developed using ReactJS and MySQL, ensuring high performance and scalability. </br>
<b> 2. No Paperwork: </b>Eliminates manual paperwork for leave applications, approvals, and tracking. </br> 
<b> 3. Accessible: </b>Accessible to all users across devices with a responsive design. </br>
<b> 4. 3-Tier Portal: </b>Designed for HoDs, faculty, and students, tailored for educational institutions. </br>
<b> 5. Secure: </b>Secured with JWT authentication and Spring Security for safe transactions. </br>
<b> 6. Cross-Platform Compatibility: </b>Fully responsive web application, compatible across all platforms. </br> 
<b> 7. Insightful Dashboards: </b>Interactive dashboards powered by Chart.js for intuitive analytics. </br>
